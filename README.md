# Writeup Finder | A writeup every day
- this python script for check **medium**,**pentester.land** writeups and etc.
## Setup
1. Go To **config.py** and set your discord **webhook** and you can change gif link
2. in ```files/lists.txt ``` you can add medium user and etc remember files content-type must be JSON/XML and fields names same in **main.py**
3. You can use  `crontab`  for run every 5 hours
